## 七、关键技术与用法（全局）

| 主题 | 说明 |
| --- | --- |
| **数据规模** | 2 亿人脸、4 亿人体 = 极大（苏州仅 1.1 亿人脸），分优先级：重点区域实时、非重点准实时 |
| **功能分工** | 图搜 → 第三方 / 短期 1 月内查询；档搜 → 情报 / 长周期 1 年 / 实战高频 |
| **上海特点** | 中间库集中市局、跨区同步慢；公安网未全覆盖、多网运行、硬件成本翻倍 |
